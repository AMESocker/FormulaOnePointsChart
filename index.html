About This Project

This interactive chart visualizes Formula 1 driver points across seasons using data from the Jolpi.ca F1 API. Built with D3.js, it enables users to explore and compare driver performance through engaging data visualization.

I developed this project to enhance my skills in data visualization, API integration, and real-time frontend development. It represents my passion for transforming raw data into intuitive, meaningful visuals that help users explore complex information more easily.
